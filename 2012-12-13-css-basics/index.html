Einbindungsmöglichkeiten

  • Element
  • HTML (Head-Bereich)
  • externe Datei (link / @import)

Element


<p style="color: #FF0000;">
    Das ist ein Absatz in roter Farbe.
</p>

HTML (Head-Bereich)


<head>
    <style>
        <!--
            p {
                color: #FF0000;
            }
        -->
    </style>
</head>

externe Datei


<head>
    <link rel="stylesheet" type="text/css" href="../css/reveal.css">
</head>

oder 


<head>
    <style>
        <!--
            @import "../css/reveal.css";
            @import url("../css/theme/reveal.css");
            @import url("../css/print/pdf.css") print;
        -->
    </style>
</head>
